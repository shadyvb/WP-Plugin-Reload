== Description ==

Simple plugin to deactivate and then activate a plugin with one click, developed mainly for Plugin developers.

The plugin adds a new 'Reload' anchor after 'Deactivate' link in plugins page.

== Installation ==

If you don't know how to use this, then you don't need it :)
